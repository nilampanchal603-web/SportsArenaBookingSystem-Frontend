# 🎨 SportsArena Frontend

Modern React frontend for SportsArena - a comprehensive sports facility and coaching management platform.

## 🚀 Features

- Browse Arenas and Sports Facilities
- Player Dashboard & Feedback Submission
- Coach Dashboard & Earnings Management
- Arena Manager Panel
- Secure Payment Integration (Razorpay)
- User Authentication

## 🛠 Tech Stack

- React.js
- Vite
- Axios
- React Router
- CSS

## 📁 Project Structure

```text
src/
 ├── assets/
 ├── components/
 ├── hooks/
 ├── pages/
 ├── router/
 └── utils/
```

## ⚙️ Environment Variables

Create `.env` file:

```env
VITE_API_URL=http://localhost:5000/api
```

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 🌐 Backend Connection

Make sure backend is running at:

```text
http://localhost:5000
```

## 📌 Notes

- Ensure backend is deployed before frontend
- Update API URL after deployment
